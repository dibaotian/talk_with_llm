# talk with llm

#### install
##### Server
- #>git clone https://github.com/dibaotian/talk_with_llm.git
- #>cd talk_with_llm/server
- #>python -m venv .venv
- #>source .venv/bin/activate
- #>python server.py

##### Client
- copy the client.py to target machine
- python client.py --host <host_ip>
